## 동적계획법 
: 복잡한 문제를 재귀를 통해 간단한 하위 문제로 분류하여 단순화하여 해결하는 방법    

#### ※ 메모제이션 
 프로그램이 동일한 계산을 반복할 때, 이전에 계산한 값을 메모리에 저장하여 동일한 계산의 반복수행을 제거하여 실행속도를 빠르게 하는 기법 


사용 조건 : 
1. 중복되는 부분분제 : 동일한 작은 문제를 반복적으로 해결해야하는 경우
2. 최적 부분 구조 : 답을 구하기 위해서 했던 계산을 반복해야하는 문제 구조 

구현방법
1. Bottom-Up 방식 : 반복문 사용 
 아래에서부터 계싼을 수행하고 누적시켜서 전체 큰 문제를 해결하는 방식
2. Top-Down 방식 : 
재귀 사용
 위에서부터 호출하여 맨 밑 상태까지 내려간 다음 결과값을 재귀를 통해 넘겨 재활용 하는 방식
